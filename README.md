# Spline-Interpolation
Spline Interpolation

В папке SplineInterpolationPart1 рассмотрен случай для кубического сплайна от одной переменной, 
рассмотрены 4 краевых условия для двух представлений: через 1-ую и 2-ую производные.
ПОстроения графиков пока что через питон, код для построения в папке SplineInterpolation. 
В папку Test  закинул скриншоты графиков, для разных функций.
Визуализацию графиков еще не доработал до финального состояния, сейчас сделаны по минимуму, для проверки кода. 

Некоторые моменты  коде еще не оптимизировал в плане компактности, еще будут доработки.

main'ы еще будут дорабатываться: в зависимости от того, какие будут подаваться входные данные.
